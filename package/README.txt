Copy the content to the folder in which W:ET is installed.
You can remove the suffix _avx or _avx2 in the filenames instead of using the default version for potentially better performance if your CPU supports AVX or AVX2.
Type in console "exec etlgm_render_high", "exec etlgm_render_medium" or "exec etlgm_render_low" depending on your GPU spec.
Disable multisampling in your GPU settings if you use etlgm_render_high or etlgm_render_medium otherwise multisampling won't work at least on my GTX 1660 Ti.
Optionally, download the RealRtCW HD texture pack from here https://drive.google.com/uc?id=10_3hGy5_UpHFxTybHAHBtf_WjS39HxAh&export=download and put it in etmain/
Source code is available at https://github.com/jumptohistory/etlegacy if you don't mind messy codes.
