Copy the content to the folder in which W:ET is installed.
You can remove the suffix _avx or _avx2 in the filenames and use those files instead for potentially better performance if your CPU supports AVX or AVX2.
Type in console "exec etlgm_render_high", "exec etlgm_render_medium" or "exec etlgm_render_low" depending on your GPU spec.
Optionally, download the RealRtCW HD texture pack here https://drive.google.com/uc?id=10_3hGy5_UpHFxTybHAHBtf_WjS39HxAh&export=download and put it in etmain/
Source codes are available at https://github.com/jumptohistory/etlegacy if you don't mind messy codes.
